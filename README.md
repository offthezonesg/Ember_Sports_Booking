# Ember Sports - Pickleball Shanghai

Shanghai's premier pickleball venue booking website.

## Features
- 8 professional pickleball courts
- Real-time availability grid
- Online booking system
- Multi-language support (English/Chinese)
- Payment integration (WeChat Pay, Alipay, Card)
- Responsive design

## Tech Stack
- HTML5
- CSS3 (Custom styles)
- Vanilla JavaScript
- Single-page application

## Deployment
- Hosting: Vercel
- Database: Supabase (planned integration)

## Development
Simply open `index.html` in a browser or run:
```bash
python3 -m http.server 8080
```
